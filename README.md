# UNP_PROJECT_BI_WITH_POWER_BI_PIZZA_DATA_ANALYSIS


I Completed a Pizza Sales and Delivery Analytics Dashboard project using Power BI to analyze overall business performance and operational efficiency.
Performed data cleaning and transformation using Power Query to ensure accurate and structured data for reporting.
Created calculated measures using DAX to analyze key metrics such as total orders, average delivery time, delay percentage, and traffic impact.
Designed interactive dashboards to evaluate monthly sales trends, customer ordering behavior, and payment preferences.
Analyzed restaurant and location performance to identify high-performing outlets and low-demand areas.
Evaluated pizza size and pizza type trends to understand customer preferences and support inventory optimization.
Developed delivery and delay analysis reports to assess operational risks and improve service efficiency.
Implemented dynamic filters and slicers to enhance user interactivity and deliver clear, actionable insights for data-driven decision-making.
This experience strenghthened my ability to translate raw data into structured dashboards that support strategic business decisions.
